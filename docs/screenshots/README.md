# 截图

将 UI 截图保存到此处，使用以下文件名（主 README 中已引用）：

| 文件 | 展示内容 |
|---|---|
| `01-home.jpg` | 通话列表（主屏幕） |
| `02-playback.jpg` | 录音播放器（进度条、声道电平、控制栏、速度） |
| `03-transcript.jpg` | 结构化转录（语气和说话人标注） |
| `04-settings.jpg` | 设置——自动录音、采样率、格式 |

## 建议

- **尺寸**：~1080×2400（标准 Android 视口）或更小——关键是可接受的质量。
- **格式**：JPG（当前截图）或 PNG（新截图推荐——更清晰，UI 边缘无压缩伪影）。
- **内容**：测试数据（使用 debug-only 的 `SeedDataActivity`），避免暴露真实号码/联系人。
- **匿名化**：即使是 debug seed 数据，也不要包含真实电话号码——使用 `+380XXX1234567` 等。
