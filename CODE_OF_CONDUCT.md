# 行为准则

本项目遵循 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。以下为关键条款的中文翻译；完整原文见链接。

## 我们的承诺

我们作为成员、贡献者和维护者承诺，让每个人参与我们社区的体验都免受骚扰，无论其年龄、体型、可见或不可见的残障、族裔、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向如何。

## 我们的标准

**有助于营造积极社区的行为示例：**

- 对他人抱有同理心和善意
- 尊重不同意见、观点和经历
- 提供建设性反馈并开放接受反馈
- 为自己的错误承担责任，并向受影响的人道歉
- 关注不仅对我们个人有利、也对整个社区有利的事情

**不可接受的行为：**

- 带有性暗示的语言或图像、人身攻击或政治攻击
- 挑衅、侮辱性评论
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息（doxxing）
- 其他在职业环境中合理视为不当的行为

## 执行

事件请报告至 `ua.lyo.su@gmail.com`。所有投诉将被保密和及时处理。

不遵守行为准则的维护者可能被暂时或永久移除项目活动参与资格。
